🎓 Student Event Management System
📌 Overview

This project is a web-based application where faculty can organize events and students can register for them.
Faculty can review student registrations and either accept or reject requests.

🚀 Features:

👨‍🏫 Faculty:
Login to the system
Create and manage events
View registered students
Accept or reject student registrations

👨‍🎓 Students:
Register and login
View available events
Apply for events
Check application status (Accepted / Rejected)

🛠️ Technologies Used:
Backend: Spring Boot
Frontend: React.js
Database: (Add your database here – MySQL / MongoDB)


🔄 Workflow
Faculty creates an event
Students register for the event
Faculty reviews registrations
Faculty accepts or rejects requests
Students can view their status


▶️ How to Run

Backend:
cd backend
mvn spring-boot:run

Frontend:
cd frontend
npm install
npm start
